About

Running Dogs

RUNNING DOGS는 서울을 기반으로 활동하는 크리에이티브 프로덕션 스튜디오입니다.
광고, 브랜드 콘텐츠, 실험적 비주얼을 넘나들며 기획과 연출, AI-native production, 포스트 프로덕션을 하나의 과정으로 연결합니다.

RUNNING DOGS is a Seoul-based creative production studio. We create commercial, branded and experimental work, bringing creative direction, AI-native production and post into one integrated process.
